    Memory Card Game

    HTML, CSS, JavaScript

    One of my first created games with JavaScript - simple memory card game where the objective is to collect the card pairs as fast as you can.     Pomodoro Time Tracker

    HTML, CSS, JavaScript, Firebase

    This pure JavaScript app was based on the traditional Pomodoro Technique to break down work/learning process to intervals of 25 minutes, separated by breaks. Helps to track your time and progress for anything you'd like. For one completed interval of 25 minutes user gets a pomodoro, the progress is being added to the total time spent. Basically, user is being notified when the time is up through browser notifications. Breaks are up to a user. Good to use for those who cares about productivity.

    Arcade Game

    HTML5 Canvas, JavaScript

    Move four directions, avoid enemies and collect crystals. Clone of classic arcade game Frogger with just HTML5 Canvas engine & JavaScript. The purpose of the game is to get as many points as possible, points are given for collecting crystals. The game can be prolonged by additional lives which appear on the map as a heart icon, however maximum of 5 lives are available at a time. But hurry up, they can disappear quickly!
